## Description

## Installation
### Firefox Add-On
Visit [Netflix 1080p UA](https://addons.mozilla.org/en-US/firefox/addon/netflix-1080p-ua/) to download add-on

### Firefox Browser
1. Run `zip -r netflix_1080p_ua.xpi manifest.json background.js content.js icon.svg popup.css popup.html popup.js`
1. Navigate to `about:debugging#/runtime/this-firefox`
2. Click `Load Temporary Add-On...`
3. Open `netflix_1080p_ua.xip`
